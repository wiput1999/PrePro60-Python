# Description
พี่ๆได้รับรหัสลับมาจากนักเปิดวาร์ปในตำนานมา ตามนี้
```
\'\"\'\"\'\"\'\"\'\"\'\
```
อยากให้น้องเขียนโปรแกรมแสดงรหัสลับด้านบนให้พี่หน่อย

| Input Specification | Output Specification |

|| string ตามที่กำหนด |
