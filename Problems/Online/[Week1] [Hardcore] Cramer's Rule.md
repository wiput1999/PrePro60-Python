# Description
!!-- MATH ALERT --!!

จงเขียนโปรแกรมขึ้นมาเพื่อหาคำตอบของระบบสมการ 3 ตัวแปร (x, y, z)
เช่นจากระบบสมการ
 
```
 1x - 2y + 3z = 7
 2x + 1y + 1z = 4
-3x + 2y - 2z = -10
```
คำตอบของระบบสมการนี้คือ x = 2, y = -1, z = 1
