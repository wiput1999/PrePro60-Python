# Description
ให้น้องๆprint รูปลูกศรตาม ด้านล่างเลยครับ

# Specification
| Input Specification | Output Specification |
| - | - |
| เลขจำนวนเต็ม ( 1 < n ) | ลูกศรตามด้านล่าง |

# Sample Case
Input : 3\
Output :
```
   *
  ***
 * * * 
   *
   *
```

Input : 8\
Output :
```
        *
       ***
      * * * 
     *  *  *  
    *   *   *   
   *    *    *    
  *     *     *     
 *      *      *      
        *
        *
```
